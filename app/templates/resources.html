Resources

We hope you had a great time at our workshops! If you'd like to keep learning and coding on your own, here are some awesome resources for you to check out, which we'll be updating from time to time. If you need any help, feel free to email us at GenCodersBCA@gmail.com or check out some of our workshops at our Github page! If you've never used Github before, simply click on the topic you want references for, click on "Clone or download" and hit "Download ZIP".


Scratch Scratch

Scratch is a visual programming language built for kids for making fun programs, games, and more!

GitHub Code Repository

The Generation Coders GitHub page houses all of our example code and projects so that you can find a lessons's starter code easily or review what you learned.

Hour of Code Block Code

Hour of Code offers a wide variety of one hour computer science tutorials for kids, from making animations of your name to building with Flappy Bird and more!

Code Combat Beginner Typed Code

Learn computer science concepts by playing a game and advancing through fun levels!

Khan Academy HTML/CSS

Khan Academy teaches not only your traditional school subjects but also has great cs courses! May be a little more advanced.

Crunchzilla Interactive JavaScript

With courses for both preteens and older, write real code and make drawings with JavaScript!

Codecademy Computer Science Courses

Codecademy has a wide range of interactive courses covering an extensive amount of topics, from HTML&CSS to Python and more!

Gameblox Block Code Game Design

Gameblox is a blocks based programming language similar to Scratch that you can use to make games, which you can then play on your phone!

App Inventor Android

App Inventor is a visual blocks based language that you can use to build an Android app in just a few hours!

Studio Code Projects

Studio Code offers a wide variey of coding projects and courses for kids beyond the shorter Hour of Code programs.
